## OG card meta tags

Drop into the `<head>` of any landing page or the GitHub repo's
`docs/` site:

```html
<meta property="og:title" content="Eventus — Event topology, made visible.">
<meta property="og:description" content="Extract the event and module topology from your JVM application. Spring Modulith first.">
<meta property="og:image" content="https://eventus.io/og-card.png">
<meta property="og:url" content="https://eventus.io">
<meta name="twitter:card" content="summary_large_image">
```
